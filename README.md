# 👶 Vicente - Baby Shower Invitation

Este proyecto es una invitación web para celebrar el baby shower de nuestro hijo **Vicente** 🍼💙

## ✨ Descripción

Se trata de una página HTML simple y elegante que permite compartir una invitación digital con familiares y amigos, incluyendo:

- Detalles del evento  
- Diseño visual amigable  
- Enlace directo para confirmar asistencia vía WhatsApp
